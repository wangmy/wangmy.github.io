Mac切换Python版本

找到安装的mac路径

系统自带的：/System/Library/Frameworks/Python.framework 自己安装的python3: /usr/local/Cellar/python@3.9/3.9.6/Frameworks/Python.framework

修改配置文件

  1. 找到根目录下的***.zshrc*** 或者 .bash_profile

  2. 修改文件 将如下内容添加到文件中,根据需求设置默认python版本

# python 配置
alias python='/System/Library/Frameworks/Python.framework/Versions/2.7/bin/python2.7'
# alias python='/usr/local/Cellar/python@3.9/3.9.6/Frameworks/Python.framework/Versions/3.9/bin/python3.9'

保存文件，在命令行执行source ~/.zshrc 或者 source ~/.bash_profile，是修改的内容生效

查看生效后的python版本

在命令行执行python --version，查看版本。
